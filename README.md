# AskRX

**AI-Powered Prior Authorization Platform — Built for DaVita Kidney Care**

AskRX streamlines the prior authorization process for dialysis and kidney care medications using IBM Watson AI — reducing administrative burden, cutting approval times, and accelerating patient access to life-critical treatments.

## 🌐 Live Platform

👉 [https://askrx.github.io/](https://askrx.github.io/)

## Features

- **IBM Watson Clinical Intelligence** — AI-driven PA review, evidence assembly, and approval recommendations
- **Regulatory Intelligence Engine** — Automatically compiles policy evidence from 12,000+ CMS, KDIGO, and payer documents
- **Real-Time Authorization Tracking** — Live dashboard for all pending, approved, and denied requests
- **Smart Document Analysis** — Automated extraction from clinical notes, lab results, and prior treatment history
- **Multi-Payer Support** — Direct submission to Aetna Medicare Advantage, UHC, Humana, Cigna, and more
- **Appeals Management** — AI-assisted appeal drafting with regulatory citations and success rate benchmarks
- **Formulary Lookup** — Real-time PA requirements, tier status, and coverage details
- **Platform Guide** — Built-in onboarding walkthrough for new users
- **HIPAA Compliant** — Audit-ready workflows, enterprise-grade security

## Platform

- Fully responsive — desktop, tablet, and mobile
- IBM Carbon Design System
- Self-contained single-page application
- Powered by IBM Watson

## Built For

**DaVita Kidney Care** — Tailored for nephrology PA workflows including ESAs, phosphate binders, calcimimetics, and dialysis-related medications.
